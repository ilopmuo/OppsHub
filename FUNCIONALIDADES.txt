TECNOLOGÍAS PRINCIPALES
------------------------
- React 19.2 (frontend)
- React Router v7 (routing)
- Supabase JS (base de datos PostgreSQL + auth en tiempo real)
- Vite 8 (build tool)
- Tailwind CSS v4 (estilos)
- Recharts (gráficas)
- Lucide React (iconos)
- React Hot Toast (notificaciones toast)
- @dnd-kit (drag-and-drop)
